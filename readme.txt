Genqiz Chatbot for WordPress lets you add AI chatbot to your website.

== Description ==

Welcome to GenQiz, where you can effortlessly create your own intelligent chatbot to enhance customer engagement and streamline your business processes. 

To get started, log in to the GenQiz platform using your Google account at https://app.genqiz.com. Once logged in, navigate to the "GEN Bots" section, where you'll begin the exciting journey of bot creation.

= Train Your Bot with AI =
Your data undergoes advanced training using state-of-the-art AI technologies like Large Language Models (LLM) and Natural Language Processing (NLP). 

You can upload text, files, links, or a FAQ CSV file with the template {question, answer}.

Through this process, your bot learns to understand your content deeply, extracting key insights and adapting its responses intelligently. 

= Bot Setting =
Configure your bot settings:
Bot Info: Name, Response Length, Description, Guidelines.
Bot UI: Display name, Icon style (close the chat to view), Widget color, Position, Size, Open by default, Title, Subtitle, Greeting, etc.

= Connect =
Easy to setup by create a bot and add token and inbox id and you can use the bot.

== Installation ==

= Automated Installation =

1. Go to your WordPress admin dashboard, then find the “Plugins” menu.
2. Click on “Add New Plugin” and search for “Genqiz”.
3. Activate the chat bot plugin through the `Plugins` menu in WordPress.
4. Go to https://app.genqiz.com and create a new account or sign-in if you already have a Genqiz account.
5. Create a new bot in Genqiz site.
6. Copy the token and inbox
7. Back Wordpress admin dashboard and click the `Genqiz` menu on the left.
8. Udpdate token and inbox
9. Save the change and your site has the bot

= Manual Installation =

1. Upload `genqiz-wordpress-plugin` directory to WordPress plugins directory (`/wp-content/plugins/`).
2. Activate the chat bot plugin through the `Plugins` menu in WordPress.
4. Go to https://app.genqiz.com and create a new account or sign-in if you already have a Genqiz account.
5. Create a new bot in Genqiz site.
6. Copy the token and inbox
7. Back Wordpress admin dashboard and click the `Genqiz` menu on the left.
8. Udpdate token and inbox
9. Save the change and your site has the bot

Zero! You don’t need to have the technical knowledge to use ChatBot. Simply drag and drop elements in our visual editor to create a chatbot.

= Can my chat bot speak multiple languages? =
Yes. You can create a multilingual bot in ChatBot. The phrases that trigger a specific interaction are fully customizable, so you can create an interaction in English that can be triggered only with the user input in English, and Spanish triggered with the phrase in Spanish.

Yes, you can customize the chat window colors, texts, and style. With the simple creator, you can adjust the chat-style and make your bot more personal.
